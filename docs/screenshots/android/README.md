# Android screenshots for GitHub README

Save your phone screenshots here with these exact filenames:

| File | Screen |
|------|--------|
| `record.png` | Record tab — balance card and transaction list |
| `add-transaction-income.png` | New transaction — Income + numpad |
| `add-transaction-expense.png` | New transaction — Expense + categories |
| `bills-overview.png` | Bills — cash flow and net balance |
| `bills-categories.png` | Bills — expense/income category donuts |
| `settings-account.png` | Settings — account, sync, appearance |
| `settings-more.png` | Settings — budget, export, about |

Then commit and push:

```powershell
git add docs/screenshots/android/
git commit -m "Add Android app screenshots for README"
git push origin main
```

**Windows tip:** After taking screenshots on your phone, copy them to this folder (USB, Google Photos, or `adb pull`).
